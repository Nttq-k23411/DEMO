1. CẤU TRÚC GIAO DIỆN NGƯỜI DÙNG

Dự án bao gồm 3 file HTML đóng vai trò là giao diện người dùng, nơi tương tác trực tiếp với các Workflow tự động

home.html: Trang chủ của website TMĐT. Chức năng chính là giới thiệu sản phẩm và dẫn hướng người dùng.

studio.html: "Xưởng Sáng Tạo AI" – đây là giao diện cốt lõi cho quá trình đồng sáng tạo. Khi nhấn nút "Tạo phác thảo", giao diện này sẽ kích hoạt Workflow 1.

cart.html: Khi khách hàng hoàn tất việc tùy chỉnh thiết kế và nhấn nút "Thanh Toán", giao diện này sẽ kích hoạt Workflow 2.
